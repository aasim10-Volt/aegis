# Changelog

All notable changes to this project are recorded here. The format follows Keep a Changelog, and the project uses semantic-style version numbers while it is in prototype.

## [0.4.1] - 2026-06-19

### Added
- Visible keyboard focus indicators across buttons, navigation, and the file list
- Accessible labels on icon-only controls (menu, search, notifications, preferences)
- Keyboard activation (Enter and Space) for icon controls
- Screen-reader announcements for toasts and dialog semantics for the modal
- Full documentation set: README, contributing guide, changelog, license, and the setup, deployment, and architecture notes
- Environment example file for the planned backend

### Notes
- No visual or behavioural changes to the existing design. This release is documentation and accessibility only.

## [0.4.0] - 2026-06-19

### Added
- Mobile and tablet layouts across every screen
- Hamburger drawer for the sidebar on small screens, with a backdrop and auto-close on navigation
- Role switcher inside the sidebar on mobile, kept in sync with the desktop switcher
- anime.js entrance animations for the login screen and for cards on every page
- Count-up animation for numeric stats

## [0.3.0] - 2026-06-19

### Added
- Working notifications panel with unread state and mark-all-read
- Command-style search with keyboard navigation
- Preferences modal
- Detail modals and toasts for every interactive surface, including stat cards, member rows, table rows, skill rows, and onboarding steps
- Animated allocation runner with live progress
- Add user, add file, and create cohort forms that update the interface
- Alert resolution that removes the alert and updates counts
- Editable scoring weights in admin settings

## [0.2.0] - 2026-06-18

### Added
- Full three-role platform prototype (student, faculty, admin)
- Cinematic boot sequence with an animated shield and system log
- Glass login screen and PDPA consent step
- Twelve pages covering dashboards, workspace, profile, onboarding, teams, alerts, allocation, users, cohorts, and settings

## [0.1.0] - 2026-06-18

### Added
- First landing-page concept for AEGIS with a maximin allocation visual
